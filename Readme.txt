1 v 1 space fighter, loosely themed on galaga

libraries required:
libsdl2-2.0-0
libsdl2-dbg(obselete on newer systems)
libsdl2-dev
libsdl2-image2.0-0
libsdl2-image-dbg(obselete on newer systems)
libsdl2-image-dev

download file with git clone https://github.com/Shriansh-Manhas/sdl_project.git
use command make
then ./game

the game begins with the ships facing each other, move quickly if you don't
want to get hit.
